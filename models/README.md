# models ディレクトリ

このフォルダには、配布者または利用者が手元の学習済みモデルを配置します。

## 推奨ファイル

- `yolo26l_face_full.pt`  
  顔モザイクアプリで使う顔検出モデル

## 任意ファイル

- `yolo26l.pt`  
  ベースモデル。再学習や比較用に使う場合のみ配置

## 注意

- `.pt.txt` などのプレースホルダではなく、**実際の重みファイル**を置いてください。
- Git には重いモデルファイルを含めない想定なので、`.gitignore` で除外しています。
- 既定以外の名前でも使えますが、その場合はアプリ起動後に `モデルパス` を変更してください。
